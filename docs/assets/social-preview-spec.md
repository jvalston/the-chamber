# Social Preview Spec (GitHub + Social Sharing)

Use this when creating the repository social preview image and README hero screenshot.

## Recommended Canvas

- Primary size: `1280 x 640` (best for GitHub social preview cards)
- Safe alternative: `1200 x 630` (common for Open Graph cards)
- Aspect ratio target: close to `2:1`

## Visual Guidelines

- Keep title text large and centered in the top/middle third.
- Avoid placing critical text at extreme edges.
- Use high contrast (dark background + bright text, or inverse).
- Include product name: `Mission Control`.
- Include one short subtitle (5 to 10 words max).

## Suggested Copy

- Title: `Mission Control`
- Subtitle options:
  - `One control plane for multi-agent operations`
  - `Run stars, comms, memory, and runtime from one cockpit`
  - `Visibility, routing, and recovery in one place`

## Export Settings

- Format: `PNG` (preferred) or high-quality `JPG`
- File size target: under `2 MB`
- Color profile: `sRGB`

## File Placement (in this repo)

- Suggested path: `docs/assets/social-preview.png`
- Optional README demo asset: `docs/assets/mission-control-demo.gif`

## Apply on GitHub

1. Open repo: `Settings`
2. Go to `General` -> `Social preview`
3. Upload `docs/assets/social-preview.png`
4. Save changes
